 🎪 SHOWPASS - Sistema de Venta de Boletas

⚙️ Requisitos
- Java JDK instalado  
- NetBeans IDE  
- Apache Tomcat 9  
- MySQL Server  

---
🗄️ Base de datos

1. Abrir MySQL  
2. Ejecutar el archivo:

   showpass.sql

Este archivo crea la base de datos y la tabla `evento` con datos de prueba.

---
 🚀 Cómo ejecutar el proyecto

1. Abrir el proyecto en NetBeans  
2. Configurar Apache Tomcat en NetBeans  
3. Importar la base de datos en MySQL (showpass.sql)  
4. Verificar conexión a MySQL en el proyecto  
5. Ejecutar el proyecto desde NetBeans  

---
 🌐 Cómo abrir el sistema

Abrir en el navegador:

http://localhost:8080/SHOWPASS-WEB/EventoServlet

---

📋 Funcionalidades

- Registrar eventos  
- Listar eventos  
- Editar eventos  
- Eliminar eventos  
- Activar / desactivar eventos
